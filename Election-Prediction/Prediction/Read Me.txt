Election-Prediction:
A Keras model to predict who will win Donald Trump or Hillary Clinton using Datasets from twitter. 


Datasets:
Datasets used are in the folder 'data'.




How to Run:
install library from Requirements.txt
To execute the code via ( Anaconda Prompt )
 just run the main.py file ==> It will call the required model for training and testing.
